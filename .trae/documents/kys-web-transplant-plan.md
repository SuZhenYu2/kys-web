## 移植架构概述

### 源项目 kys-cpp 核心模块
根据源码分析，kys-cpp 包含以下核心模块：

| 模块 | 功能 | 优先级 |
|------|------|--------|
| `src/Engine.cpp` | SDL2 封装、渲染引擎 | 高 |
| `src/Scene.cpp` | 场景管理、切换 | 高 |
| `src/BattleScene*.cpp` | 各类战斗系统 | 高 |
| `src/Object.cpp` | 游戏对象、角色 | 高 |
| `src/Audio.cpp` | 音频系统 | 中 |
| `src/TextureManager.cpp` | 贴图管理 | 高 |
| `mod/BattleMod.cpp` | 战斗规则配置 | 高 |
| `local/` | 资源文件 | 高 |

### Web 移植需要替换的技术栈

| C++ 组件 | Web 替代方案 |
|---------|-------------|
| SDL2 图形 | Canvas 2D / WebGL |
| SDL2 音频 | Web Audio API |
| bass 音频库 | Howler.js / Web Audio |
| Lua 脚本 | JavaScript (原生) |
| 本地文件系统 | IndexedDB / Fetch API |
| 网络对战 | WebSocket |
