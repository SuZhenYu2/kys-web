## 一、游戏概述

### 1.1 游戏定位

**"活着的金庸武侠世界"**

这是一款以金庸武侠小说为背景的大型多人在线角色扮演游戏（MMORPG）。游戏的核心特色在于引入了基于 DeepSeek 大模型的 AI NPC 系统，使游戏世界中的每一个 NPC 都拥有独立的性格、记忆、作息和社交能力。玩家不再是独自探索江湖的孤独旅人，而是与张三丰、郭靖、黄蓉、杨过等传奇人物共同生活在一个动态的、有生命力的武侠世界中。

### 1.2 核心卖点

| 卖点 | 描述 |
|------|------|
| **AI 驱动的江湖** | 每个 NPC 都有记忆和情感，会记住你、评价你、与你互动 |
| **真实的武侠人生** | 作息、修炼、社交、恩怨，体验完整的江湖生活 |
| **开放的世界事件** | 江湖大事由玩家和 NPC 共同书写 |
| **深度武功系统** | 从基础招式到绝世神功，修炼路径清晰可见 |

### 1.3 目标用户画像

| 用户类型 | 特征 | 需求 |
|----------|------|------|
| **金庸粉丝** | 熟悉原著，热爱武侠文化 | 还原度高、剧情丰富、能与经典角色互动 |
| **MMORPG 玩家** | 追求社交和竞争 | 门派战、排行榜、装备收集 |
| **休闲玩家** | 时间有限，追求放松 | 挂机功能、自动寻路、轻度肝度 |
| **剧情党** | 重视故事体验 | 主线剧情、支线丰富、有选择分支 |

### 1.4 技术指标

| 指标 | 目标值 |
|------|--------|
| 支持同时在线 | 5000+ |
| 服务器 Tick Rate | 20Hz（移动同步）、10Hz（战斗）、1Hz（AI） |
| 地图最大实体数 | 单屏 200 个实体（玩家+NPC） |
| AI 对话响应时间 | P99 < 3s |
| 客户端帧率 | 60 FPS |

## 二、世界观设定

### 2.1 历史背景

**时间线：南宋理宗景定四年（公元 1263年）**

这一年，射雕英雄传故事已过二十余年，神雕侠侣的故事刚刚落幕，襄阳大战后郭靖夫妇名震天下，蒙古铁骑再次南下，江湖风云涌动。各路高手纷纷出山，少林、武当、丐帮、明教等门派暗中较量，无数青年才俊踏入江湖，寻求武功秘籍、争夺武林至尊的称号。

#### 2.2.1 中原武林（★★★★☆危险度）

**襄阳城**
- 描述：南宋边防重镇，郭靖黄蓉驻守之地
- 重要NPC：郭靖、黄蓉、郭芙、武敦儒、武修文
- 功能：主城、安全区、军事任务发布
- 特色：
  - 郭靖每日在城头巡视（18:00-19:00）
  - 黄蓉在府邸处理丐帮事务
  - 军需官处可接守城任务

**终南山**
- 描述：全真教总坛所在，钟南毓秀

- 功能：奇门遁甲、桃花岛武功

- 特色：

**华山**

## 三、角色系统

### 3.1 基础属性详解

| 属性 | 全称 | 作用 | 影响范围 |
|------|------|------|----------|
| **气血 (HP)** | 生命值 | 生命，降至0则死亡 | 战斗生存、野外存活 |
| **内力 (MP)** | 真气值 | 使用武功消耗 | 技能释放、内功威力 |
| **力道** | 外功攻击 | 物理伤害基础 | 普通攻击、外功伤害 |
| **根骨** | 外功防御 | 物理减伤基础 | 受到的外功伤害 |
| **身法** | 速度与闪避 | 出手顺序、闪避概率 | 先手、回避、韧性 |
| **悟性** | 学习效率 | 武功学习、升级经验 | 技能成长、奇遇触发 |
| **福缘** | 运气属性 | 暴击、掉落、奇遇 | 稀有获取、随机事件 |

### 3.2 属性计算公式

```
// 战斗属性计算
外功伤害 = (力道 × 技能系数 + 武器伤害) × (1 + 暴击率 × 暴击伤害加成)
内功伤害 = (内力 × 内功系数) × (1 + 内功加成)
物理减伤 = 根骨 / (根骨 + 对方力道 × 0.5) × 85%
闪避率 = 身法 / (身法 + 对方身法 × 0.8) × 60%
暴击率 = (福缘 - 50) / 500 + 装备暴击率 + 武功暴击加成
先手值 = 身法 + 随机(0, 10) + 装备先手加成
```

### 5.2 伤害计算详细公式

```
// 普通攻击伤害
基础伤害 = 攻击方力道 × 0.5 + 武器伤害
最终伤害 = 基础伤害 × 技能系数 × (1 + 暴击加成) × (1 - 对方减伤率)

// 武功伤害
武功伤害 = (攻击方力道 + 内力 × 0.3) × 技能威力系数 × 熟练度加成 × (1 - 防御减伤)

// 绝学伤害（特殊武功）
绝学伤害 = 基础伤害 × 绝学系数 × 境界加成 × 情绪/环境修正

// 伤害类型
外功伤害：受力道、根骨、防御影响
内功伤害：受内力、悟性、内功等级影响
混合伤害：力道和内力各占50%
真实伤害：无视防御（部分绝学）
```

```sql
    -- 记忆内容
    content TEXT NOT NULL,
    summary VARCHAR(500),  -- 自动生成的摘要
    importance INT DEFAULT 5,  -- 重要性 1-10
    
    -- 关联信息
    related_player_id BIGINT,  -- 关联玩家（可为null）
    related_npc_id BIGINT,    -- 关联NPC（可为null）
    related_event_id BIGINT,  -- 关联事件
    
    -- 时间戳
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    expires_at TIMESTAMP,  -- 仅对瞬时/短期有效
    
    -- 索引
    INDEX idx_npc_type (npc_id, memory_type),
    INDEX idx_related_player (npc_id, related_player_id),
    INDEX idx_expires (memory_type, expires_at)
);

-- NPC 关系表
CREATE TABLE npc_relationship (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    npc_id BIGINT NOT NULL,
    player_id BIGINT NOT NULL,
    
    -- 关系值
    favorability INT DEFAULT 0,  -- 好感度 -100 ~ 100
    trust INT DEFAULT 0,         -- 信任度 0 ~ 100
    impression TEXT,             -- 对玩家的印象（AI生成文本）
    
    -- 特殊标记
    is_friend BOOLEAN DEFAULT FALSE,
    is_enemy BOOLEAN DEFAULT FALSE,
    is_mentor BOOLEAN DEFAULT FALSE,
    
    -- 最后交互时间
    last_interaction_at TIMESTAMP,
    
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    
    UNIQUE KEY uk_npc_player (npc_id, player_id)
);
```

### 7.2 AI 动态任务生成

#### 7.2.1 任务生成规则

```yaml
任务生成条件:
  触发源:
    - NPC当前情绪（高兴/悲伤/愤怒/担忧）
    - NPC当前处境（缺钱/有人骚扰/需要帮助）
    - 世界事件（蒙古入侵/江湖大事）
    - 玩家关系（信任度高可能请求帮助）
  
  限制条件:
    - 任务难度与玩家等级匹配
    - 任务奖励与难度平衡
```

```protobuf
enum MessageType {
    // 登录相关
    LOGIN_REQUEST = 1001;
    LOGIN_RESPONSE = 1002;
    
    // 移动相关
    PLAYER_MOVE = 2001;
    PLAYER_MOVE_ACK = 2002;
    ENTITY_MOVE_BROADCAST = 2003;
    
    // 战斗相关
    BATTLE_START = 3001;
    BATTLE_ACTION = 3002;
    BATTLE_END = 3003;
    
    // 对话相关
    DIALOGUE_START = 4001;
    DIALOGUE_INPUT = 4002;
    DIALOGUE_RESPONSE = 4003;
    
    // NPC相关
    NPC_STATE_UPDATE = 5001;
    NPC_INTERACT = 5002;
    
    // 社交相关
    CHAT_MESSAGE = 6001;
    FRIEND_REQUEST = 6002;
    TEAM_INVITE = 6003;
}

// 玩家移动消息
message PlayerMoveRequest {
    int64 player_id = 1;
    Position target = 2;
    MovementType type = 3;  // WALK, RUN, MOUNT
    int64 timestamp = 4;
}

message EntityMoveBroadcast {
    int64 entity_id = 1;
    EntityType entity_type = 2;  // PLAYER, NPC, MONSTER
    Position from = 3;
    Position to = 4;
    MovementType type = 5;
    int64 timestamp = 6;
}

// AI对话请求
message AIDialogueRequest {
    int64 npc_id = 1;
    int64 player_id = 2;
    string player_input = 3;
    DialogueContext context = 4;
}

message AIDialogueResponse {
    int64 npc_id = 1;
    string response = 2;
    string emotion = 3;
    repeated string actions = 4;
    int64 timestamp = 5;
}
```
