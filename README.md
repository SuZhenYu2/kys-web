# 武侠江湖 - Wuxia RPG Game

一个基于 Phaser.js 开发的武侠 RPG 游戏，参考金庸群侠传设计。

## 游戏特色

- 🎮 **经典武侠 RPG** - 探索江湖世界
- ⚔️ **回合制战斗系统** - 与敌人战斗
- 🏘️ **开放世界地图** - 自由探索
- 🗣️ **NPC 对话系统** - 与角色互动
- 📱 **响应式设计** - 支持多种屏幕

│   ├── objects/                   # 游戏对象

## 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式运行

```bash
npm run dev
```

游戏将在 `http://localhost:3000` 启动。

### 构建生产版本

```bash
npm run build
```

构建产物将输出到 `dist/` 目录。

## 游戏操作

### 移动和探索
- **WASD** 或 **方向键** - 移动角色
- **空格键** - 与 NPC 对话/触发战斗
- **ESC 键** - 打开/关闭菜单

### 战斗系统
- **[1]** - 打开武功列表
- **[2]** - 使用物品
- **[3]** - 普通攻击
- **[4]** - 逃跑
- **空格键** - 选择并使用武功
- **数字键 1-4** - 选择武功
- **ESC** - 返回战斗菜单

### 菜单系统
- **B** - 打开背包
- **S** - 打开武功面板
- **C** - 查看角色状态

## 游戏设计

### 游戏循环
1. 探索地图
2. 与 NPC 对话
3. 学习武功或购买物品
4. 遭遇敌人
5. 回合制战斗
6. 获得经验和银两
7. 升级和购买装备
8. 挑战更强的敌人

### 属性说明
- **生命 (HP)** - 生命值，归零则战斗失败
- **内力 (MP)** - 使用武功消耗，归零无法使用高级武功
- **攻击力** - 影响造成的伤害
- **防御力** - 减少受到的伤害
- **速度** - 影响战斗行动顺序
- **银两** - 购买物品和学习武功的货币
- **经验值** - 积累足够升级
